Building the Sample

In Visual Studio Ultimate 2013:

  1. Start Visual 2013 and from the menu, select File > Open > Project/Solution.
  2. Browse to the directory that contains the unzipped sample files. Double-click the Visual Studio Solution (.sln) file.
  3. From the menu, select Build > Build Solution.

To run the sample in the Visual Studio debugger, from the menu select Debug > Start Debugging. To run the sample in Visual Studio 2013 Ultimate without debugging, from the menu select Debug > Start Without Debugging.
