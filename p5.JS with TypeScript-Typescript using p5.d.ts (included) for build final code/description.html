Introduction

Building the Sample

Run Visual Studio in Admin Mode. Yes you need to download NuGet  (Requirement for ASP.net project) Microsoft.CodeDom.Providers.DotNetCompilerPlatform.

Make Default.aspx as "Start Page". Use local IIS. Setup site.

Description

I used https://code.msdn.microsoft.com/windowsapps/p5JS-with-TypeScript-171eeca7/https://github.com/Gaweph/p5-typescript-starter approach for build this application.

This project wrapped with VS C# web application, you can easily move everything to HTML file. Just copy and paste html section to a different file.

On some reason, “GitHub” version does not work for me. I fixed some common errors and add Class for Sketch.  Main recommendation use “Namespace”, in this case, you do not need “import”.

Each public class/value must use “export”. Always use explicit accessibility to class member – private, public, protected.

This project includes all p5 libraries and p5.d.ts file:

This is the p5 instance constructor. A p5 instance holds all the properties and methods related to a p5 sketch. It expects an incoming sketch closure and it can also take an optional node parameter for attaching the generated p5 canvas to a node.

The sketch closure takes the newly created p5 instance as its sole argument and may optionally set preload(), setup(), and/or draw() properties on it for running a sketch.
