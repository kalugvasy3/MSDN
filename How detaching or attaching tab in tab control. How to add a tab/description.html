Introduction

From Visual Studio, please RUN in Debug Mode or you may comment out some code which identifies the existing instance of this process.  If you run the second process you recieve some messeges (just click OK). You will be able to comment out messeges yourself. It is very helpful for a debugin porpuse which includes multiply window.

The second process sends messege to the first process to open new Window and after that kill itself.

 

How to detached or attached tab in the tab control.  How to add a tab from the different window (in the same process).

How to send the message to existing process to open a new window.
